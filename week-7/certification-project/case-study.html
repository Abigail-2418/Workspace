Case Study: Catan Field Guide

A reflection on design decisions and implementation

Project Overview

The Settler's Field Guide to Catan is an interactive documentation website designed to teach new players the mechanics and strategies of the board game Settlers of Catan. The project was built with a focus on creating a reference manual aesthetic while maintaining modern web design principles and full responsiveness across devices.

Why I Chose This Topic

Personal Connection

Settlers of Catan is one of the most popular strategy board games, introducing millions of players to modern tabletop gaming. The game's mechanics are straightforward yet deep, making it an ideal subject for structured documentation that could genuinely help new players learn.

Design Challenge

Creating a documentation-style website presented unique design challenges compared to typical marketing or portfolio sites. The content needed to be dense and informative while remaining visually engaging and easy to navigate. This required balancing information density with white space, hierarchy, and visual interest.

Technical Goals

This project served as an opportunity to demonstrate proficiency in React, React Router for multi-page navigation, Tailwind CSS for responsive styling, and creating a cohesive design system. The sidebar navigation and smooth scrolling sections required thoughtful state management and UX consideration.

Design Decisions

Color Palette Strategy

The color scheme was directly inspired by Catan's resource system. Each resource has a distinct color in the game, and I incorporated these into the design to create thematic coherence:
The dominant emerald green serves as the primary color, evoking the island setting while providing excellent contrast for readability. Each section uses different accent colors to create visual distinction without breaking cohesion.

Information Hierarchy

Each section follows a consistent structure: icon + heading, introductory paragraphs, highlighted information boxes, ordered or bulleted lists, and supporting images. This predictable pattern helps users scan content efficiently while the visual variety maintains engagement throughout the long-form documentation.

Decorative Elements

Hexagon shapes appear as background decorations throughout the design, directly referencing Catan's hexagonal game tiles. These elements were intentionally kept subtle (low opacity, outline-only) to enhance atmosphere without competing with content readability.
The gradient backgrounds transition between resource colors, creating visual flow while maintaining the field guide's cohesive aesthetic.

Layout Structure

Desktop Layout

On desktop viewports (1024px and above), the sidebar navigation remains fixed on the left side, providing constant access to all sections. This design pattern is common in documentation sites because it allows users to jump between sections without scrolling back to the top.

  • Sidebar: Fixed, 320px width, full height
  • Main Content: Left margin 320px, full width
  • Navigation: Active state highlighting, smooth scroll

Mobile: Collapsible Menu

On mobile devices, the sidebar transforms into a hamburger menu that slides in from the left. This preserves screen real estate while maintaining full navigation functionality. The menu automatically closes when a section is selected, providing smooth UX flow.

  • Sidebar: Slides in/out, overlay mode, z-index layering
  • CTA Buttons: Stacked vertically instead of horizontal
  • Decorative Elements: Reduced for performance

Smooth Scroll Implementation

Clicking navigation items triggers smooth scrolling to the corresponding section. This creates a polished, modern experience while maintaining the feel of a single-page application. The active section updates based on scroll position, providing visual feedback about the user's current location in the document.

Typography and Color Choices

Font Strategy

The design uses the default system font stack provided by Tailwind CSS, which renders as San Francisco on macOS/iOS, Segoe UI on Windows, and Roboto on Android. This ensures optimal readability and performance without requiring custom font loading.

  • H1 for hero titles
  • H2 for section headings
  • Paragraphs for body text

What I Would Improve

Interactive Elements

The current design is informational but largely static. Adding interactive components would enhance learning: a dice roll probability calculator, an interactive board generator, or a resource trading simulator. These tools would transform the guide from passive reading to active learning.

Search Functionality

As documentation grows, search becomes essential. Implementing a client-side search (using libraries like Fuse.js) would allow users to quickly find specific rules or strategies without manually navigating sections. This would particularly benefit users returning to refresh specific knowledge.

Custom Illustrations

While the current design uses stock photography effectively, custom illustrations showing game mechanics (hex tile placement, road networks, trading examples) would significantly improve clarity. Visual diagrams of abstract concepts like "Longest Road" or "resource production" would enhance comprehension.

Accessibility Enhancements

  • Keyboard navigation for the sidebar menu
  • RIA labels for decorative elements and interactive components
  • Skip-to-content link for screen reader users
  • Color contrast verification for all text combinations
  • Focus indicators for all interactive elements

Reflection & Takeaways

Building the Settler's Field Guide to Catan was an exercise in balancing dense informational content with engaging visual design. The project reinforced the importance of consistent structure, thoughtful color application, and responsive design patterns that work across devices.

This project demonstrated that good documentation design is about more than organizing information—it's about creating a reading experience that feels purposeful, navigable, and visually coherent. The skills developed here apply directly to technical documentation, educational platforms, and any content-heavy web application.
